# Spiral Exponential Function Visualizer
Interactive Visualization of Spiral Exponential Trajectories

This repository provides an interactive Python-based visualizer for trajectories
generated by the **Spiral Exponential Function**, a solution of the
**Spiral Exponential Equation (SEE)**.

It is designed as a research and visualization tool for exploring the geometric
structure of exponential motion in the complex plane, focusing on the fixed
angle between position and velocity vectors.
This tool provides an intuitive way to understand how exponential growth and
rotational behavior are continuously connected through a single geometric
parameter.

## Features
Visualization of spiral exponential trajectories in the complex plane.  
Interactive control of the angle Œ∏ between position and velocity vectors.  
Interactive control of the time parameter t.  
Real-time display of the trajectory f(t).  
Visualization of the current position as a point.  
Optional visualization of derivative directions:
- Position vector
- Velocity direction (Œ∏)
- Acceleration direction (2Œ∏)
- Jerk direction (3Œ∏)
- Snap direction (4Œ∏)

All derivative vectors are displayed according to the geometric rotation rule
prescribed by the Spiral Exponential Equation.

You can install all dependencies using pip:

## Quick Start
Clone the repository and run the visualizer:
```bash
git clone https://github.com/KoheiAbeLab/spiral-exponential-visualizer
cd spiral-exponential-visualizer
python spiral_exponential_visualizer.py
```

## Usage
Run the visualizer:
python spiral_exponential_visualizer.py

Controls:
Sliders ‚Üí adjust Œ∏ and t
Checkboxes ‚Üí toggle visibility of vectors and higher-order derivatives
